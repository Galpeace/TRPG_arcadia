# Witnessed — Artifact 仕様書

---

## ① キャラクタービューア（ローカルHTMLファイル）

**性質：** ローカルファイル。Claude Code が生成する。閲覧専用。  
**保存先：** `characters/[名前]/sheet_view.html`  
**生成タイミング：** キャラクターを保存するとき（セッション終了時）  
**テンプレート：** `templates/character_view_template.html`（このファイルの{{プレースホルダー}}を実データで置換して出力する）

ブラウザで開くと現在のキャラクター状態が美しく一覧できる静的HTMLである。  
編集機能は持たない。データの変更は sheet.md を編集したうえで再生成する。

### 含むセクション

| セクション | 内容 |
|---|---|
| キャラクター基本情報 | 名前・種族・ジョブ・現在地・目的 |
| 状態 | 負傷・状態異常・疲労 |
| スキル | アクティブ・パッシブ・ユニーク・称号（各テーブル） |
| 所持品 | テーブル形式 |
| 進行中のクエスト | テキスト |
| 評価 | 評価者・評価内容のリスト |
| NPC関係リスト | 名前・関係・態度の簡易リスト |
| 舞台設定 | 最終セッションの5項目 |
| メタ情報 | 生成日・死亡フラグ（死亡済みの場合は赤いバナー） |

### NPC関係リスト の形式

```
衛兵隊長 ガルン ／ 知人・中立やや好意的 ／ 王都西門担当。顔を覚えている
宰相の密偵 不明 ／ 敵対・監視中 ／ 尾行を確認済み
```

シンプルな `名前 ／ 関係 ／ メモ` の1行形式。後で相関図に発展させてよい。

### デザイン仕様

ダークファンタジー調（下記 CSS 変数を基準とする）：

```css
--bg: #18181c;
--surface: #22221e;
--border: #383430;
--text: #ddd8cc;
--muted: #887e6e;
--accent: #b08848;
```

フォント：`'Hiragino Kaku Gothic Pro', 'Meiryo', sans-serif`  
最大幅：900px、中央寄せ。外部ライブラリ不使用。  
死亡済みの場合：赤いバナーを最上部に表示し、全体に暗い赤みのスタイルを適用する。

## ② セッション開始ツール（Claude Chat アーティファクト）

**性質：** Claude chat 上で動作するインタラクティブ HTML アーティファクト。  
**役割：** セッション開始時のキャラメイク情報を入力し、GM チャットへ貼れる整形テキストを出力する。  
**制約：** ローカルファイルとの連携は不可。WQ データはハードコード。出力はコピー&ペーストのみ。

アーティファクトの使用は任意。使用しない場合はテキストで直接入力してよい。

### セクション構成

#### STEP 1：方向性選択（カード4枚）

| カード | ラベル | 説明 |
|---|---|---|
| A | GM裁量起点 | キャラと舞台を設定し、クエストはGMに委ねる |
| B | WQ選択 | WQを選ぶ。舞台設定の一部はWQから自動取得 |
| C | クエスト方向性 | やりたいことを一言で提示 |
| D | 継続キャラクター | 保存済みシートを貼り付け、起点モードを選択 |

選択に応じて後続セクションが表示/非表示を切り替える。

#### WQセクション（B モード時のみ）

4枚のカードから選択。クリックで3項目（支配者・緊張・トーン）が自動入力される（read-only 表示）。

**WQ データ（ハードコード）**

| ID | タイトル | 支配者（自動） | 緊張（自動） | トーン（自動） |
|---|---|---|---|---|
| WQ-01 | 「王冠の暗夜」 | 腐敗した宰相の影響下にある王国（王は病床） | 王女への脅威、権力集中、信仰と政治の癒着 | 陰謀と退廃（王都）／ 素朴で不穏（辺境） |
| WQ-02 | 「六星の影が南へ」 | 辺境領主または無政府（ヴェレン） | 森から来る何か・魔物増加・工作員の出没 | 素朴で不穏（辺境）／ 荒廃と生存（ヴェレン） |
| WQ-03 | 「帝国の楔」 | 山賊王朝・ウラリス・メジプトのいずれか | 外来勢力の浸透・旧秩序の崩壊・住民の選択 | 荒廃と生存・終末的 |
| WQ-04 | 「二つの王冠の間で」 | 名ばかりの領主（帝国の影響下の辺境貴族） | 利権争い・移民との軋轢・よそ者への不信 | 素朴で不穏・陰謀と退廃（混在） |

WQ 選択後にプレイヤーが入力：土地の詳細・PCの所在理由

#### 継続キャラセクション（D モード時のみ）

- sheet_view.html のコピーテキスト、または手動テキストを貼り付ける textarea
- 注記：「世界はこのキャラクターを記憶しない。死亡済みは継続不可。」
- サブモード選択：A / B / C

#### キャラクターフォーム（A/B/C、または D サブ選択後）

| フィールド | タイプ |
|---|---|
| 名前・種族・ジョブ | text |
| 目的 | text |
| アクティブスキル | 最大6行（ランク＋名前＋備考） |
| パッシブスキル | 行追加/削除（名前＋内容） |
| ユニークスキル | 行追加/削除（ランク＋名前＋制約） |
| 称号 | 行追加/削除（称号＋効果〈空欄可〉＋由来）。異名と称号スキルの統合軸 |
| 所持品 | 行追加/削除（品名＋数量＋備考）。通貨はゴールド（G） |

#### 舞台設定（自由記述）

舞台設定は固定フォームではなく**自由記述（textarea）**とする。
以下の観点を手がかりとして表示するが、入力は文章で構わない。

- 観点ヒント：土地の類型 ／ 支配者 ／ くすぶる緊張 ／ トーン ／ PCの所在理由
- A / C モード：上記を手がかりに自由記述
- B モード：支配者・緊張・トーンはWQから自動取得し read-only 表示。
  プレイヤーは「土地の詳細」と「PCの所在理由」を自由記述で補う。

#### クエスト方向性フィールド（C モードのみ）

一言テキスト入力。例：`失踪した兄を探したい`

#### コピーボタン

「📋 セッション開始テキストをコピー」→ 下記フォーマットをクリップボードへ。

### コピー出力フォーマット

```
──────────────────────────────────────────
【Witnessed セッション開始】
──────────────────────────────────────────
■ 方向性
B. WQ選択 — WQ-01「王冠の暗夜」（アルカディア王国）

■ キャラクター
名前：〇〇　種族：〇〇　ジョブ：〇〇
目的：〇〇
アクティブスキル：
  ・スキル名（ランク） — 備考
パッシブスキル：
  ・スキル名：内容
ユニークスキル：
  ・スキル名（ランク）：制約
称号：
  ・称号名（効果があれば記載／空欄＝評判のみ）：由来
所持品：
  ・品名（×数量）備考
  ・所持金：〇〇 G

■ 舞台設定（自由記述）
（プレイヤーが書いた文章をそのまま出力）
※ B モードでは冒頭に「支配者／緊張／トーン：（WQから自動取得）〇〇」を併記する。

■ クエスト方向性（Cモード時のみ）
〇〇
──────────────────────────────────────────
```

D モード（継続キャラ）の場合は「■ キャラクター」の前に貼り付けたシートテキストを挿入し、「※ 世界はこのキャラクターを記憶しない。NPCは前セッションのPCを知らない。」を付記する。

### デザイン

アーティファクト①と同じ CSS 変数を使用。モードカードは選択状態で色が変わる（各モードに異なるアクセントカラーを使用）。
