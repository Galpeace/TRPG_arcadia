

VaultフォルダでPowerShellを開き、以下を実行する。

  

```powershell

Get-ChildItem -Recurse -Filter *.md |

Where-Object {

    $_.FullName -notmatch "\\.obsidian\\" -and

    $_.FullName -notmatch "\\.git\\" -and

    $_.Name -ne "merged.md"

} |

Sort-Object FullName |

ForEach-Object {

    "`n`n# $($_.BaseName)`n"

    Get-Content $_.FullName -Encoding UTF8

} | Set-Content merged.md -Encoding UTF8
