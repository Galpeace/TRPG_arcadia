# 第1部　概観

## 1.1 基本情報

- 名称：ヨプの大森林
- 位置：アルカディア大陸北東部
- 性質：アルカディア王国を環状に取り囲む広大な森林
- 形状の由来：地理的な偶然。環状の形をした森林がもともと存在しており、アルカディア王国はその内側を天然の防壁として利用して栄えた

## 1.2 森林の二重の機能

ヨプの大森林はアルカディア王国にとって二重の機能を持ちます。

**外敵に対する防壁**

森林越えは困難であり、外部からの侵攻を実質的に阻んでいます。王国が長期にわたって安定した秩序を保てた地理的な理由はここにあります。

**北方ヨウィンとの断絶**

同時に、北方ヨウィン地方とアルカディア王国を切り離す障壁でもあります。森林をはさんで南北に隣接しながら、両者は経済的にも政治的にも実質的に断絶しています。ヨウィンがアルカディア王国から独立した地域として存在できているのは、この森林が交易と支配の双方を困難にしているためです。

# 第2部　森林の特性と地理

## 2.1 規模と構造

- アルカディア王国を完全に環状に囲む
- 幅は場所によって異なるが、徒歩での横断には数日〜数週間を要する
- 東側が比較的薄く（伐採事業の対象）、西側の奥深い部分が最も濃密

## 2.2 魔素濃度の勾配

- 森林全体が高めの魔素濃度を持つ
- 外縁部：人が定期的に往来できる程度
- 中間部：冒険者が危険を感じるレベル
- 中心部・西側奥部：人類には極めて危険なレベル
- 禁足地（最西端）：人類未踏に近い濃度

## 2.3 アクセス

王国へのアクセスは街道か船のみに限られます。

**街道**

- 複数の幹線街道が森林を貫く
- 主要な街道は定期的に整備され、比較的安全
- 脇道は危険
- 夜間の通行は推奨されない

**東海岸ルート**

- 現在、現国王による伐採事業と港湾整備が進んでいる
- 伐採がほぼ海岸まで到達しており、港湾村が新設された

# 第3部　森林内の主要地域

## 3.1 街道沿い

- 定期的に整備されている
- 宿場、見張り小屋が点在
- 比較的安全だが、魔物の出没はある
- 商人、旅人、冒険者が利用する

## 3.2 中間部

- 街道から外れると急速に危険度が増す
- ダンジョンが散在する
- エルフの集落が点在するが、外部者を歓迎しない
- 魔物の縄張りが複雑に入り組む

## 3.3 中心部・西側奥部

- 魔素濃度が極めて高い
- 強力な魔物が生息する
- エルフの諸集落の深部
- 通常の冒険者には実質的に立ち入り不能

## 3.4 禁足地（最西端）

- 竜王国・フェルガード連邦以西の深部
- 人類未踏の地に近い
- 伝説のエルフ都市国家ユグドラシルがあるとされる
- 立ち入った者はほとんど帰還しない

## 3.5 東側（伐採対象区域）

- アルカディア現国王による伐採事業が進行中
- 現在、伐採は海岸線近くまで到達している
- 森林東端の海岸沿いに、伐採作業員を中心とした村が最近新設された
- 開拓が進むにつれ、魔物の移動や生態系の変化が生じている
- 伐採作業員と魔物の衝突が頻発

----

# 第4部　エルフの諸集落

## 4.1 集落の概要

森林内には複数のエルフ集落が点在します。各集落は独立しており、統一的な政府や指導者は存在しません。

- 規模は小さく、数十〜数百人程度
- 森林との共生を基本とする
- 外部者（特に人類）への態度はさまざまだが、基本的に距離を置く
- 深部に近い集落ほど閉鎖的

## 4.2 外部との関係

**人類圏との関係**

- ほとんど交流がない
- エルフは人類を避けており、交易も基本的に成立しない
- 街道沿いでエルフを見かけることはまれ
- 伐採事業には強く反発している

**魔王国との関係**

- 森林が事実上の防壁となっており、直接的な接触は少ない
- 魔素の変化などを通じた間接的な影響は感じている

## 4.3 伝説のユグドラシル

禁足地の奥にエルフの都市国家ユグドラシルがあるとされます。

- 蘇生効果を持つ世界樹を中心とした樹上都市
- 詳細は謎に包まれている
- 現存するかどうかも不明
- エルフの各集落との関係も不明

# 第5部　政治的問題（伐採事業）

## 5.1 現状

アルカディア現国王ハインリヒは、王国東海岸の港湾確保のため、森林東端の伐採事業を推進しています。

- 目的：東海岸への直接アクセスによる海洋交易の拡大
- 規模：森林東端の限定的な伐採
- 進行度：伐採が海岸線近くまで到達。港湾村が新設された段階

## 5.2 対立の構図

**伐採推進側**

- 国王ハインリヒ：港湾確保による交易拡大を重視
- 利権を求める貴族：伐採利権と新設港湾による利益を期待
- 商人ギルドの一部：交易拡大を歓迎

**伐採反対側**

- エルフの諸集落：生活圏および森林の破壊に強く反発
- 環境・信仰を重視する勢力：自然破壊への懸念
- 改革派貴族の一部：外交リスクを懸念

**中立・日和見**

- ポートリア海洋国：既存の交易関係への影響を注視
- 多くの貴族：利権と外交リスクを天秤にかける

## 5.3 PCへの影響

- 伐採作業員の護衛依頼
- エルフとの接触（交渉や衝突）
- 伐採により追い出された魔物への対処
- 利権争いへの巻き込まれ

# 第6部　危険と探索

## 6.1 魔物

- 外縁部：ゴブリン、ウルフ、コボルドなど
- 中間部：トレント、ウィル・オ・ウィスプ、ビーストなど
- 深部：フォレストドラゴン、古代精霊、森の主など（Sランク相当）

## 6.2 ダンジョン

- 旧時代の遺跡
- エルフの放棄した神殿
- 魔物の巣窟化した廃城
- 魔素の濃い地点に自然発生したダンジョン

## 6.3 迷子リスク

- 魔素の影響による方位磁針の狂い
- 街道を外れると帰還が困難になる
- 経験豊富なガイドの同行が推奨される

# 第7部　物語類型と運用上の指針

## 7.1 ヨプの大森林を舞台にした物語類型

**1. 街道の旅**

- 王都カディアへの往来
- 旅人・商人の護衛
- 街道上の魔物・盗賊との対峙

**2. 伐採事業をめぐる物語**

- 東側新設村での生活と事件
- エルフとの衝突・接触
- 港湾利権をめぐる争い

**3. 森林探索**

- ダンジョン攻略
- エルフ集落への接触試み
- 禁足地への挑戦

## 7.2 PCの想定類型

- 街道を旅する商人・冒険者
- 伐採作業員・護衛の傭兵
- 森林の調査を依頼された冒険者
- エルフとの接触を試みる外交的人物
