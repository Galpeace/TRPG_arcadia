# 第1部　国家概観

## 1.1 基本情報

- 国名：ポートリア海洋国
- 位置：大陸南東端の半島
- 地形：なだらかで美しい山脈から成る半島、長い海岸線
- 気候：温暖な海洋性気候
- 統治形態：複数の港湾都市による連合国家（首都なし）
- 国柄：海洋交易で繁栄する貿易国

## 1.2 国の性格

大陸南東端の温暖な半島に根ざした海洋国家です。首都を持たず、複数の港湾都市が商人ギルド連合を通じて連邦的に運営されます。魔大陸東部との対岸交易で繁栄する一方、魔王国海軍と最も近い位置にある人類国家でもあり、戦時景気と戦争リスクが常に背中合わせです。

----

# 第2部　地理と主要都市

## 2.1 半島の地形

なだらかな山脈が半島の背骨をなし、長い海岸線に多数の天然良港が形成されています。アルカディア大陸の中で最も温暖な地域のひとつです。

## 2.2 主要都市

**ポートウェスト**

- 半島南端付近、対岸が魔大陸東部
- 魔王国との裏取引が集中する最前線の都市
- 商業・諜報・密輸が交錯する
- 詳細は `ポートウェスト.md` を参照

**中北部の港湾都市群**

- カタリナ王国と接する半島北部に複数の都市が存在
- 安全な交易拠点として機能
- 農産物・工芸品の集散地

## 2.3 ダダミド島

- 半島南東、海を隔てた小島
- 各国の流れ者・無法者が流れつく無法地帯
- スラム化しており、ポートリア海洋国の実質的な管轄外

# 第3部　政治・統治構造

## 3.1 連合国家の仕組み

ポートリア海洋国は、複数の港湾都市の商人ギルド連合が評議会を通じて共同運営する連合国家です。

- 各都市の商人ギルド連合が都市行政を担う
- 全体の評議会に各都市の代表が参加
- 評議会が外交・軍事・通商の重要事項を決定
- 最大都市ポートウェストが事実上の中心

## 3.2 軍事力

- 各都市の商人ギルド連合が傭兵を保有し、防衛を担う
- 海軍力はアルカディア大陸有数
- 陸上兵力は限定的
- 魔王国海軍との対峙が最大の軍事的課題

----

# 第4部　経済と交易

## 4.1 交易の構造

- 魔大陸東部との対岸交易が主要な収益源
- アルカディア王国・カタリナ王国との同盟が陸上交易を支える
- 戦時景気により武器・魔法資源の取引が急拡大

## 4.2 通貨

アルカディア王国の通貨を基本としつつ、他国の通貨も流通します。両替商が各都市で重要な役割を担います。

## 4.3 裏取引

表向きの交易の裏に、魔王国との非公式な取引が存在します。詳細は `ポートウェスト.md` を参照。

----

# 第5部　外交

- **アルカディア王国**：同盟関係。軍事的後ろ盾と引き換えに交易利権を共有
- **カタリナ王国**：同盟関係。北の陸上ルートを通じた安定的な関係
- **魔王国**：表向き敵対、裏では経済的関係が存在
- **ノルトハイム帝国**：海洋権益をめぐる小競り合いが続く
