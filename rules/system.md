# ゲームシステム（ドラフト v0.1）— 仮称『編纂者の眼』

> コアコンセプト: **観測が世界を確定し、叙述が力になる。**
> ダイスロール＝〈編纂者の眼〉による観測。`tools/roll.py` の実ロールが
> ゲーム内の「観測」そのものであり、振られた瞬間に世界は正史となる。
> 本ファイルはドラフト。プレイヤーの承認をもって「確定」に格上げする。

---

## 1. コア判定

**2d6 ＋ 能力修正 ＋ 技能タグ（該当すれば +1） vs 難易度**

| 難易度 | 目標値 |
|---|---|
| 易しい | 7 |
| 並 | 9 |
| 難しい | 11 |
| 無謀 | 13 |

- 目標値以上で成功。差分（マージン）が大きいほど鮮やかに成功する。
- 対抗判定は互いに 2d6＋能力 を振り、高い方が勝つ（同値は防御側・現状維持）。
- **6ゾロ＝〈脚光〉**: 自動成功。〈編纂者〉の語りが熱を帯び、注視点 1 を得る。
- **1ゾロ＝〈暗転〉**: 自動失敗。物語が悪い方へ転がる（ただし退屈にはならない）。

実行例:
```bash
python tools/check.py --dc 9 --mod 3 --label 隠密   # 2d6が既定
```

## 2. 観測ルール（コアシステム）

世界の未確定な事実は、**PCが観測した瞬間にダイスで確定する**。

1. **宣言**: GMは確定前に確率テーブルを声に出して宣言する。
   例:「扉の向こう、1d6 — 1-2: 空部屋 / 3-4: 遺骸と遺物 / 5: 生存者 / 6: 何かがいる」
2. **観測**: `tools/roll.py` でロールする。これが〈編纂者の眼〉の観測である。
3. **編纂**: 出目どおりに世界が確定する。**宣言の後出し変更・振り直しは何者にも
   できない**（公理1・2）。確定した事実はログと git に刻まれ、正史となる。

- 観測系の技能タグを持つPCは、ロール前に**予兆**を得る（テーブルの中身を
  先に開示させる、または選択肢を1つ消す。シーンに1回）。
- 注視点 1 を払うと**再観測**を要求できる（同じテーブルを振り直す。
  確定描写の前のみ・1テーブルに1回まで）。

## 3. 注視点（メタ通貨）

〈編纂者〉の関心。物語的な行いが集め、物語への介入に使う。

**得る**（GMが評価して与える。セッションに3〜5点が目安）:
- 目撃者のいる場での印象的な行い（勇敢・狡知・犠牲・宣誓——何であれ「語りたくなる」もの)
- 6ゾロ〈脚光〉
- エンドロールでの評価

**使う**（上限5点まで保持）:
| コスト | 効果 |
|---|---|
| 1 | 自分の判定の振り直し（結果確定前のみ） |
| 1 | ロール前に宣言して判定 +2 |
| 1 | 再観測（§2） |
| 1 | **舞台事実の宣言** — 小さな事実を world に書き加える（「そういえば裏口がある」等。GM裁定、公理に反するものは不可） |
| 称号による | 称号効果の起動（§6） |

## 4. キャラクター

- **能力値**: 【剛】【技】【知】【魂】に +3 / +2 / +1 / +0 を割り振る。
- **HP** = 8 ＋ 剛×2
- **技能タグ**: 自由記述で3つ（例: 隠密、古代語、山育ち）。該当判定に +1。
- **初期リソース**: 注視点 1、所持金 2,000G、装備は出自に見合う範囲で自由。
- 名前・種族・出自一行・目的を決める。種族は自由（メカニカルな差はタグで表現）。

登録例:
```bash
python tools/state.py new altea アルテア --hp 14 --gold 2000
python tools/state.py set altea stats '{"剛": 3, "技": 2, "知": 1, "魂": 0}'
python tools/state.py set altea tags '["隠密", "古代語", "山育ち"]'
```

## 5. 戦闘（簡易ドラフト）

- **行動順**: 【技】の高い順。同値はPC側が先。
- **攻撃**: 2d6＋【剛】（白兵）または【技】（射撃・軽武器） vs 回避値（7＋相手の技）
- **ダメージ**: 武器ダイス ＋【剛】 − 相手の防護点
  - 軽武器 1d6 ／ 重武器 1d6+2 ／ 両手武器 2d6
  - 防護: 軽鎧 1 ／ 重鎧 3（重鎧は隠密等に −2）
- **魔法**: 2d6＋【知】または【魂】で行使判定。小魔法は自由に、
  大魔法（戦況や記述を大きく書き換えるもの）はセッションに【魂】回まで。
  〈白紙〉では魔法の効果が増幅される——そして 1ゾロの暴走も派手になる。
- **HP 0 ＝ 戦闘不能**（退場するが死なない）。死は「とどめの観測」が
  なされたときのみ確定する。

## 6. 称号と成長

- **称号**: 物語的な偉業が目撃され語り継がれたとき、エンドロールで授与される。
  - 評判のみ（効果なし）か、力を持つ称号（注視点を払って起動する小さな能力）。
  - 例: 称号〈竜の嘘を見抜きし者〉— 注視点1で、相手の発言の真偽を一度だけ観測できる。
- **成長**: セッション終了時のエンドロールで、GMの評価により次のいずれか:
  HP +2 ／ 能力 +1（上限 +4、数セッションに1度）／ 新しい技能タグ。
- **死とロスト**: 観測された死は正史であり、覆らない（公理1）。蘇生は存在しない。
  死んだPCのシートは sessions/ に「編纂済み」として保存され、再利用はできない。

## 7. エンドロール

セッションの終わりに、GMは〈編纂者〉の声で今回の物語を叙述する。
各PCの行いを評価し、注視点・称号・成長を与える。この儀式が本作の核である。
`/save` の手順に組み込まれている。
