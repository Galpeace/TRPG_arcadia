# ゲームシステム（ドラフト案B v0.1）— 仮称『万神の桟敷』

> コアコンセプト: **主役は何者でもありうる。** ネズミでも、伝説の剣豪でも、
> 悪の死霊術師でも——システムは「強さ」ではなく「物語」を測る。
> 本ファイルはドラフト。プレイヤーの承認をもって「確定」に格上げする。

## 1. キャラクター — 存在・真髄・格

**能力値は存在しない。** キャラクターは次の3つで記述される。

- **存在**: あなたが何者かの一行宣言。完全に自由。
  例:「王宮の床下に住む年経たネズミ」「頂を極めて隠遁した伝説の剣豪」
  「死者の安寧を説く、世に言う悪の死霊術師」
- **真髄**: その存在を定義する性質を3つ、自由記述で。力にも弱みにもなる。
  例（ネズミ）: 「どこにでも入り込む」「人語を解する」「臆病、しかし好奇心が勝つ」
  例（剣豪）: 「剣に懸けて嘘を言わぬ」「殺気を読む」「過去に斬った者の影を背負う」
- **格**: 世界における物理的・社会的なスケール。1〜5。
  格1: 鼠・子供 ／ 格2: 並の大人 ／ 格3: 市井の達人 ／ 格4: 国に聞こえた強者 ／ 格5: 伝説

**初期リソース**: 命脈 10（全員一律）、喝采 1、所持金は存在に見合う額（GM裁定）。

```bash
python tools/state.py new nezu チュー助 --gold 0
python tools/state.py set nezu essence '["どこにでも入り込む", "人語を解する", "臆病、しかし好奇心が勝つ"]'
python tools/state.py set nezu rank 1
```

## 2. 判定 — 困難さは「その存在にとって」測る

**2d6 ＋ 該当する真髄1つにつき+1（最大+3） vs 難易度**

| 難易度 | 目標値 |
|---|---|
| その存在には少し難しい | 7 |
| その存在には難しい | 9 |
| その存在には至難 | 11 |
| その存在には無謀 | 13 |

- **お手のもの**: 存在から自明にできることは判定不要で成功する。
  剣豪は並の兵を判定なしで打ち倒し、ネズミは判定なしで物陰に消える。
- **難易度は絶対値ではない。** その存在にとっての物語的困難さで測る。
  - ネズミ:「台所から食料を盗む」判定不要 /「眠る猫の傍を抜ける」並9 /「人間に意図を伝える」至難11
  - 剣豪:「賊徒の一団を退ける」判定不要 /「百人組手」至難11 /「剣を抜かずに戦を止める」無謀13
- **6ゾロ＝〈万雷〉**: 自動成功。桟敷が沸き、喝采 1 を得る。
- **1ゾロ＝〈野次〉**: 自動失敗。だが喝采 1 を得る——神々は華麗な転落も愛でる。
- 対抗判定は互いに振って高い方（同値は防御側・現状維持）。

```bash
python tools/check.py --dc 9 --mod 2 --label 猫の傍を抜ける
```

## 3. 格差ルール — 戦えない相手とどう戦うか

- **直接の力比べで格差2以上があるとき、判定は発生しない。上が通る。**
  ネズミ（格1）は騎士（格3）と斬り合えない。剣豪（格5）は野盗（格2）に後れを取らない。
- ただし格が守るのは「力比べの土俵」だけである。**搦め手・機転・物語の土俵に格は無い。**
  ネズミは騎士に勝てないが、兜の中に飛び込むことはできる。鎧紐を齧ることも、
  彼の秘密を寝言から聞き出すこともできる。
- **見せ場**（1幕に1回、喝采2）: 宣言したその行動について、格差による自動敗北を
  無効化し、判定に +3。ネズミが竜の目を潰す瞬間、神々は総立ちになる。

## 4. 喝采（メタ通貨）

神々の熱。物語が沸かせ、運命を撓ませるのに使う。**上限5。**

**得る**（GMが桟敷の反応として与える。1幕に3〜5点が目安）:
- 真髄に殉じた選択（損だと分かっていて、その存在らしく振る舞う）
- 大胆な賭け、鮮やかな機転、悲痛な犠牲——善悪を問わず「沸く」場面
- 6ゾロ〈万雷〉、1ゾロ〈野次〉
- しがらみの受諾（§5）

**使う**:
| コスト | 効果 |
|---|---|
| 1 | 自分の判定の振り直し（結果確定前のみ） |
| 1 | ロール前に宣言して判定 +2 |
| 1 | **演出介入** — 場に小さな事実を書き加える（「換気口がある」「衛兵は顔見知りだ」等。GM裁定） |
| 2 | **見せ場**（§3。1幕に1回） |

## 5. しがらみ — 真髄は弱みでもある

GMは真髄や存在を逆手に取った提案ができる。
（「臆病が、その一歩を踏み出させない」「死霊術師の貴方を、村人は決して信じない」）

- 受け入れれば**喝采 1** を得て、物語はその方向へ転がる。
- 拒否するなら喝采 1 を払う（持っていなければ拒否できない）。

## 6. 命脈と退場

- **命脈は全員一律 10。** これは肉体の頑健さではなく「物語への居座る力」である。
  鼠も伝説も、主役である限り等しく粘る。
- 争いの応酬: 対抗判定で敗れた側が差分だけ命脈を失う（最低1）。
  武器や鎧の優劣は数値でなく、真髄と描写——つまり判定の有利不利として扱う。
- **命脈 0 ＝ 舞台袖へ**: 倒れ、捕えられ、流される——退場の危機。死ではない。
- **死**は、命脈0の者への「幕引きの一撃」が描写されたときのみ確定する。
  死んだPCは星座に編まれる。**ロストであり、再演はない。** ただしその名と演目は
  world/ に星座として永遠に記録される。
- 幕間（場面の大きな転換）で命脈は全快する。傷は残っても、物語は続く。

## 7. カーテンコール（セッション終了の儀式）

幕が下りるとき、GMは桟敷のざわめき——神々の批評——を語り、各PCを評価する。

1. 今回の演目を短く叙述する（どの場面で桟敷が沸いたか）
2. 評価に応じて: 喝采の持ち越し付与 ／ **二つ名**の授与（口上師が語り広める）／
   成長 — 真髄の追加（最大5）または書き換え、まれに**格の上昇**（演目の格が上がる大事件のみ）
3. プレイログを書き出してコミットする
