# ギュンター

## 基本情報

- **名前**：ギュンター
- **年齢**：50代
- **性別**：男性
- **職業**：鉱夫の酒場の主人
- **所在地**：ヴェルニース

## 外見

- がっしりとした大柄な体格
- 短く刈った灰色の髪
- 豪快な笑顔
- 傷跡が残る腕
- エプロン姿

## 性格

- 豪快で陽気
- 面倒見が良い
- 話好き
- 元狩人としての誇り
- 鉱夫や狩人たちの良き相談相手

## スキル

- 料理（狩人のロースト）：A
- 酒場経営：A
- 狩猟知識：B（元狩人）
- 社交術：B

## 経歴

- ヴェルニースで生まれ育つ
- 若い頃は名うての狩人だった
- 怪我をきっかけに狩人を引退
- 酒場を開業
- 鉱夫や狩人たちの社交場として繁盛している

## 人間関係

- **エレナ**：若き狩人、元狩人の先輩として助言することも
- **ブルーノ**：鉱夫のリーダー、友人
- **ヴェルニースの鉱夫・狩人たち**：常連客

## 所持品・資産

- 鉱夫の酒場（建物）
- 調理道具一式
- 酒の在庫
- 多少の蓄え
- 元狩人時代の武器（飾り）

## 特記事項

- 名物料理は「狩人のロースト」（狩猟肉のロースト、森のキノコ添え）
- 元狩人としての経験から、狩人たちにアドバイスすることも
- 酒場は村の社交の中心
- 豪快な性格で、村人から慕われている

## 物語の種

- 酒場での交流
- 元狩人時代の思い出
- エレナへの助言
- 村の人間関係の調整役
