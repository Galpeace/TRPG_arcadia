## 使い方と免責

### 強度目安について

本図鑑の強度目安（D〜S）は**個体の平均的な基準**である。以下の要素で個体差が生まれる。

- マナ濃度の高い地域で生まれた個体は強い
- 長命・多経験の個体は平均より高ランク
- 群れ・集団行動の場合は実効的な脅威が跳ね上がる
- 「史上最強のゴブリン」がSランク相当になることは公理上ありえる

GMはこの目安を出発点として、セッション状況に合わせて自由に調整してよい。

### 知性レベルの定義

| 表記 | 内容 |
|---|---|
| 本能 | 捕食・縄張り・逃走のみ。言語なし |
| 低 | 簡単な意図の疎通は可能。罠を仕掛ける程度の知恵 |
| 中 | 集団行動・簡単な戦術・道具使用。交渉は困難 |
| 高 | 言語・目的・感情を持つ。交渉・欺瞞・取引が可能 |
| 人類同等 | 人類と区別のつかない知性。「魔族」とみなされる領域 |

### 魔族について

「魔族」は学術分類ではなく世間の呼称である。高知性・人型のモンスターをおおまかに指す。  
彼らは社会・政治・目的・感情を持ち、**PCと交渉・同盟・対立できる存在**である。  
自動的に敵対する理由はない。すべては状況整合性で決まる。

---

### 低知性モンスターの発言表現

知性レベルが「低」以下のモンスターが言語を発する場合、**カタカナ交じりの表現**を用いる。  
文法は崩れ、語彙は限られ、感情や本能がそのまま言葉になる。

| 知性レベル | 発言スタイル | 例 |
|---|---|---|
| 本能 | 発言なし。唸り声・咆哮のみ | 「グルルル……」「キィィッ」 |
| 低 | カタカナ語・単語レベル。文をなさない | 「コロス、コロス！」「ニゲル！」「エサ、エサ！」「イタイ！」 |
| 中 | 片言・不完全な文。語彙が増えるが訛りが残る。一部カタカナ | 「お前たち、囲メ。逃がすな」「ここは、オレたちの場所」 |
| 高〜人類同等 | 通常の言語。訛り・癖・方言はあってよい | （通常描写に準じる） |

知性が「中」の個体は、怒り・恐怖・命令のとき一時的にカタカナが混じることがある。  
GMは知性の上昇（進化）を発言の変化で表現してよい。

## 種族進化の原則

モンスターは以下の要因で進化する（rulebook 第13節参照）。

```
経験 — 戦闘・捕食・知識の蓄積
環境 — 高マナ地帯・特殊気候・呪われた土地
身体変化 — 変異・感染・大量のマナ摂取
評価 — 群れの中で頭角を現した、恐れられた
達成 — 強敵を倒した、長期間生き延びた
```

進化は**一方向ではない**。分岐する。  
ゴブリンが剣術を磨けばゴブリン・ウォーリアーへ、魔法を磨けばゴブリン・メイジへ進化する。  
最終的にはゴブリン・キングという個体支配型の頂点に至ることもある。

## ◆ 低脅威帯（強度目安 D〜C）

---

### ゴブリン

**強度目安：** D〜C  
**生息域：** 森林縁・廃屋・洞窟・ダンジョン低層  
**形状：** 1〜1.2m。緑褐色の皮膚。小柄だが素早い  
**知性：** 低〜中  

**代表スキル（個体によって異なる）**
- 短剣術（D）：粗雑だが速い刺突。集団で挟めば危険
- 投石（D）：射程は短いが奇襲で有効
- 群れ戦術（パッシブ）：3体以上で連携すると行動が統制される

**進化系統**

```
ゴブリン（D〜C）
  ├─ ゴブリン・ウォーリアー（C）── 戦闘特化。体格も大きい
  ├─ ゴブリン・メイジ（C）────── 魔法に目覚めた個体。距離を取って戦う
  ├─ ゴブリン・シャーマン（C）── 儀式・毒・呪いを扱う。群れに影響を与える
  └─ ゴブリン・ロード（B）────── 複数の群れを統率する上位個体
       └─ ゴブリン・キング（A）── 広域支配・個体として人類の精鋭に匹敵
```

**GMメモ：** 単体では大した脅威にならないが、10体・20体の群れは別の話。PCが「ゴブリン狩り」で慢心すると痛い目を見る構図が王道。知性が上がるほど罠・奇襲・交渉が絡んでくる。

---

### スライム

**強度目安：** D〜C  
**生息域：** 洞窟・地下水路・廃屋・湿地帯  
**形状：** 不定形。色は種別によって異なる（透明・緑・赤・黒等）  
**知性：** 本能〜低  

**代表スキル**
- 酸溶解（C）：接触した有機物・金属を溶かす。強度は個体差が大きい
- 分裂（パッシブ）：大きなダメージを受けると二分割されることがある
- 形態変化（パッシブ）：狭い隙間をすり抜ける

**進化系統**

```
スライム（D〜C）
  ├─ メタルスライム（C）────── 金属質の外皮を発達させた。物理攻撃をほぼ無効化
  ├─ アシッドスライム（C）── 酸溶解が極めて強力。接近禁忌
  ├─ ポイズンスライム（C）── 毒を分泌する。接触で毒状態
  └─ キングスライム（B〜A）── 大量のスライムが合体した巨大個体。強度は合体数に比例
```

**GMメモ：** 基本スライムは弱いが「倒し方がわからない」局面でPCを詰める使い方ができる。メタルスライムは物理無効→魔法・炎・水が有効という攻略工夫の典型例。

### アンデッドの自然発生について

> この節はスケルトン・グール両種に共通する前提知識である。

高マナ地帯に**死体を放置する**と、骨格や肉体がマナを吸収し、自然にアンデッドとして発生することがある。  
マナ濃度が高いほど発生が速く、強い個体が生まれやすい。

**発生の分岐：肉が残っているかどうか**

```
放置された死体
  ├─ 肉が残っている（フレッシュな状態）→ グール系として発生
  └─ 肉が腐落し骨格のみになった状態  → スケルトン系として発生
```

**予防方法**

| 方法 | 効果 | 備考 |
|---|---|---|
| 聖水を注ぐ | マナの浸透を阻害する。発生を遅らせるか防ぐ | 定期的な再施行が必要な地域もある |
| 火葬 | 最も確実。骨ごと灰にすれば再発生しない | 燃料・時間・人手が必要 |
| 教会の弔い儀式 | 聖水と同効果〜それ以上。地域・教義によって差がある | 高位神官ほど効果が高い |

**世界観上の意味：** 貧しい農村や戦場は死体の処理が追いつかず、アンデッドの巣になることがある。  
「弔い」の習慣は宗教的な意味だけでなく、生存上の必要性から根付いている。

### スケルトン系

**強度目安：** D〜C  
**生息域：** 古戦場・墓地・ダンジョン・高マナ地帯の廃村  
**形状：** 乾いた骨格。武器・防具を装備していることがある  
**知性：** 本能（命令を受け取れる。自発的な意思決定はほぼしない）  

**スケルトンの特性**  
肉が腐落した後に骨格がマナを吸収して発生する。**空腹を持たない**。生者を積極的に追うのではなく、縄張りを守る・もとの習慣に従って徘徊するといった受動的な行動をとる。  
生前が戦士なら武器を構えて立ち、農民なら農具を持って歩き回ることがある。

**代表スキル**
- 武器術（D〜C）：生前の記憶の残滓。かつての技能を保持する個体がある
- 不死者の耐性（パッシブ）：毒・病・睡眠・恐怖は無効。斬撃・刺突は効きにくい。打撃・炎・光魔法に弱い
- 骨の再結合（パッシブ）：バラバラにされてもマナが残れば時間とともに再結合することがある

**進化系統**

```
スケルトン（D〜C）
  └─ スケルトン・ウォーリアー（C〜B）── 武装・強化。元凄腕戦士の骨は即Bランク相当
       └─ ボーン・ロード（B）──────────── 高マナ地帯で長年蓄積した骨の支配者。周囲のスケルトンを統率する
```

**GMメモ：** 死霊術PCにとって「資源」になりうる存在。大量のスケルトンを操る戦術は強力だが、PCの評価に直接影響する（村人は怯え、教会は敵視し、一部の傭兵は距離を置く）。スケルトン自体は攻撃しなければ反応しないことも多く、「通り抜けられるか」という緊張感の演出にも使える。

### グール系

**強度目安：** C〜B（発生直後はD〜C）  
**生息域：** 高マナ地帯・廃村・古戦場・地下水路・魔王国周辺  
**形状：** 肉が残った人型。腐敗しかけの皮膚・長く発達した爪。動きは見かけによらず素早い  
**知性：** 低〜中（腐肉・生者への渇望が行動の核）  

**グールの特性**  
フレッシュな死体（肉が残っている状態）がマナを吸収して発生する。  
発生直後は緩慢でほぼ反射的にしか動かず、世間では「**ゾンビ**」と呼ばれることが多い。  
しかし捕食とマナ吸収によって肉体が変質し、急速に動きが速くなり、知性が芽生えてくる。  
**スケルトンとの最大の違いは「空腹がある」こと。** グールは常に何かを食べようとしている。

**代表スキル**
- 爪撃（C）：速い連続攻撃。防御が薄い部位を本能的に狙う
- 腐肉耐性（パッシブ）：毒・病は無効。炎・光魔法に弱い
- 生命感知（パッシブ・C）：生者の体温・鼓動を感知する。暗所や霧の中でも追ってくる

**進化系統**

```
ゾンビ（D〜C）── グール発生直後の俗称。緩慢・低知性。腐肉を食べることで成長する
  └─ グール（C〜B）── 知性と速度が戻ってくる。狩りに戦術を使い始める
       └─ ワイト（B）── 恐怖を帯びた負のオーラを持つ。接触で生命力を吸収する。知性は人類近い
            └─ リッチへの道（極めて稀）── グール系が魔法を習得した場合、リッチへの自力進化が起こりうる
```

**GMメモ：** 「ゾンビだと思って油断したらグールだった」という誤認は王道のホラー構図。動き・速さ・目の焦点で段階を表現してよい。知性があるワイトは交渉の余地があることもある（人間だったころの記憶が一部残っている個体）。

リッチ（A〜S）── 魔法使いが死後も術を行使するために自ら不死化した存在（別項参照）。グール系と異なり、意図的な変容である。

---

### ウルフ

**強度目安：** D〜C  
**生息域：** 森林・平原・山岳  
**形状：** 大型の狼。マナの影響で自然の狼より大きい場合が多い  
**知性：** 本能〜低  

**代表スキル**
- 牙（C）：深い裂傷を与える。首・腱を狙う本能的な戦術
- 群れ狩猟（パッシブ）：3頭以上で挟撃・追い込みを本能的に行う
- 追跡（B）：嗅覚と持久力による長距離追跡

**進化系統**

```
ウルフ（D〜C）
  ├─ ブラッドウルフ（C〜B）── 血の匂いに敏感。負傷したPCを執拗に追う
  ├─ ダイアウルフ（C〜B）──── 馬ほどの巨体。騎乗可能（魔族などが乗ることがある）
  └─ フェンリル（A〜S）────── 伝説的な巨大白狼。魔力を帯び、嵐を呼ぶとされる
```

**GMメモ：** 遭遇時点では普通の狼でも、PCが負傷した瞬間にブラッドウルフが覚醒するような演出が可能。

## ◆ 中脅威帯（強度目安 C〜B）

---

### オーク

**強度目安：** C〜B  
**生息域：** 森林縁・ヴェレン・廃村・山岳  
**形状：** 2〜2.5m。灰緑色の皮膚、巨体、粗削りな武具を自作する  
**知性：** 中〜高  

**代表スキル**
- 打撃（C〜B）：単純だが重く、防御を崩す
- 頑強（パッシブ）：痛みへの耐性が高い。中傷程度では行動を止めない
- 戦争叫び（B）：周囲の仲間の士気を高める。敵の恐怖判定に影響

**進化系統**

```
オーク（C〜B）
  ├─ オーク・ウォーリアー（B）── 戦闘特化。重装備。正面突破の申し子
  ├─ オーク・シャーマン（B）──── 部族の呪術師。属性魔法・呪い・儀式
  └─ オーク・ロード（A）──────── 強者が集うオーク部族の長。PCでも苦戦する
```

**GMメモ：** ゴブリンが「数」の脅威なら、オークは「質」の脅威。1体でも侮ると痛い。知性があるため交渉・脅迫・取引が成立する場面も多い。

---

### トロール

**強度目安：** B〜A  
**生息域：** 洞窟・湿地・深い森  
**形状：** 3〜4m。緑灰色の皮膚、長い腕。鈍重だが圧倒的な力  
**知性：** 低〜中  

**代表スキル**
- 叩きつけ（B）：地面・壁に叩きつける。中傷〜重傷を一撃で与えうる
- 再生（パッシブ・B）：傷が急速に修復される。炎と酸は例外（再生を阻害）
- 投擲（C）：岩を遠距離に投げる

**進化系統**

```
トロール（B〜A）
  └─ アシュドロール（A）── 老成・大型化したトロール。再生がさらに速い
```

**GMメモ：** 「炎や酸を確保する」という攻略工夫の典型例として使いやすい。知性が低いため純粋な力押し勝負になりがち。罠・環境を使わないと消耗戦になる。

---

### ハーピー

**強度目安：** C〜B  
**生息域：** 山岳・海岸の断崖・廃塔  
**形状：** 女性の上半身と大きな翼・鉤爪の下半身。高速飛行が可能  
**知性：** 中〜高  

**代表スキル**
- 鉤爪攻撃（C）：急降下からの引っ掻き。距離を詰めさせない
- 歌声（B）：聴いた者を行動不能または誘惑状態にする（精神干渉）
- 飛行（パッシブ・B）：高度な機動力。地上戦闘が有利なPCには脅威

**GMメモ：** 「歌声を封じる（耳を塞ぐ、対呪術）」か「飛行を封じる（弓・魔法）」かという攻略工夫が生まれやすい。知性が高いため交渉・取引の余地もある。

---

### リザードマン

**強度目安：** C〜B  
**生息域：** 湿地・水辺・温暖地域。人間社会に定住する個体も存在する  
**形状：** 人型。鱗に覆われた2m前後の体躯。竜の末裔を自称する  
**知性：** 高〜人類同等  

**代表スキル**
- 鱗の防御（パッシブ・C）：生来の甲冑。物理攻撃に高い耐性
- 竜の息吹（C〜B）：個体によって炎・毒・冷気等を持つ。使用頻度は少ない
- 尾撃（C）：意外に強力な強打。バランス崩しに有効

**GMメモ：** 「魔族」として人間社会に溶け込む個体も多い。差別の対象でもあり、PCとの関係次第で強い盟友にも仇敵にもなる。竜の末裔という自負が行動の動機になりやすい。

## ◆ 高脅威帯（強度目安 B〜A）

---

### リッチ

**強度目安：** A〜S  
**生息域：** マナ濃度の高い廃墟・塔・地下迷宮  
**形状：** 白骨化した魔法使い。魔力による身体保持。見かけによらず素早い場合がある  
**知性：** 人類同等以上（生前の知性を保つ。むしろ死後に研ぎ澄まされる）  

**代表スキル**
- 死霊魔法（S）：大規模な不死者召喚・操作。呪いの付与。魔力の吸収
- 古の知識（パッシブ）：生前の研究蓄積。魔法・歴史・弱点に関する情報量が膨大
- 魔力核（ユニーク）：本体は魔力核（フィロクテリー）に宿る。核を破壊しない限り完全には死なない

**GMメモ：** 「魔力核がどこにあるか」という謎解きが攻略工夫の核になる。直接戦闘ではまず勝てない相手として設計できる。交渉・取引の余地もある（長命で孤独なため、知的な会話相手を求めていることがある）。エルダー・リッチはさらに上の存在（六星クラス）。

---

### サキュバス / インキュバス

**強度目安：** B〜A  
**生息域：** 都市・宮廷・魔王国  
**形状：** 美しい人型。翼・尾・角を持つが、隠せる個体もある  
**知性：** 人類同等〜それ以上  

**代表スキル**
- 媚術（A）：視線・声・香りによる精神干渉。抵抗力の低い相手は即座に籠絡される
- 影の移動（B）：暗所間を瞬時に移動
- 生命力吸収（A）：接触により対象の活力・魔力・精神力を奪い取る

**GMメモ：** 直接戦闘よりも情報戦・心理戦に強い。PCが「騙された」と気づく瞬間が核心。正面から戦わせない設計にすると面白い。六星クラス（ネモフィラ）は別格。

---

### ドラゴン

**強度目安：** A〜S  
**生息域：** 山頂・ドラゴン山脈・マナ高濃度地帯  
**形状：** 巨大な四肢翼竜。体色は個体によって異なり、能力に関連することが多い  
**知性：** 高〜人類同等（老齢になるほど賢くなる）  

**代表スキル**
- 竜の息吹（S）：炎・冷気・毒気・雷等（個体で異なる）。範囲・威力ともに圧倒的
- 鱗の鎧（パッシブ・A）：ほとんどの物理攻撃を弾く
- 竜の恐怖（パッシブ・B）：視線・咆哮だけで周囲の生物に恐怖状態を与える
- 飛行（パッシブ・A）：高速飛行。騎馬での追跡は不可能

**進化系統**

```
ドレイク（B）── 小型のドラゴン。飛べないが素早い
  └─ ドラゴン（A〜S）── 成体。竜の息吹・飛行が揃う
       └─ エルダードラゴン（S）── 古代個体。知性は人類以上。魔法も使う
```

**GMメモ：** 正面からSランク相当のPCでなければまず勝てない。「交渉する・弱点を突く・罠にはめる・騎乗モンスターを利用する」など攻略の余地を設計しておく。老ドラゴンは知性が高く、長命ゆえに世界の歴史を知っている。情報源・同盟先としての使い道もある。

### 悪魔（デーモン系）

**強度目安：** B〜S（位階によって大きく異なる）  
**生息域：** 魔王国・高マナ地帯・召喚師によって呼び出された場所  
**形状：** 人型〜異形。位階が高いほど人型に近づく傾向  
**知性：** 中〜人類同等以上  

**代表スキル（位階により差異あり）**
- 炎の爪（C〜A）：悪魔火による攻撃。通常の水では消えない
- 契約術（ユニーク）：人間と契約を結び、代償と引き換えに力を貸す
- 転移（B）：短距離の瞬間移動

**進化系統**

```
インプ（D〜C）── 下位悪魔。使い魔として扱われることが多い
  └─ デーモン（B〜A）── 中位悪魔。戦闘・契約ともに強力
       └─ 上位悪魔（A〜S）── 個体名を持つ独立した存在。六星クラスの悪魔も存在
```

**GMメモ：** 「契約」という要素がロールプレイの核になりやすい。悪魔との取引は代償が必ず伴い、その代償が後からPCを苦しめる展開は王道。

## ◆ 動物モンスター（軽度脅威帯）

> 公理11節：すべての動物はマナの産物であり、進化の潜在能力を持つ。

通常の動物（オオカミ・クマ・大蛇等）は自然界に存在するが、マナ濃度が高い地域では「変異した動物モンスター」として強化された個体が出現する。

| 元の動物 | 変異後 | 強度目安 |
|---|---|---|
| オオカミ | ブラッドウルフ | C〜B |
| クマ | 魔熊（マジックベア） | B |
| 大蛇 | ナーガ / 竜蛇 | B〜A |
| 鳥 | ロック / 雷鳥 | B〜A |
| ネズミ | ラットキング（群れ統率個体） | B |

**GMメモ：** 農村PCが「普通のクマだと思っていたら魔熊だった」という局面は、油断した際の難易度調整として機能する。変異は外見的な異常（大きさ・目の色・傷の付き方）でわかることが多い。

## 脅威帯早見表

| 強度目安 | 主な種 | PCへの脅威感 |
|---|---|---|
| D | ゴブリン・スライム・スケルトン初級 | 普通の戦闘員が対処可能 |
| C | ゴブリン・メイジ・オーク・ウルフ | 単体でも油断すれば危険 |
| B | オーク・ロード・トロール・ドレイク | 準備なし正面突破は危険。攻略が必要 |
| A | ゴブリン・キング・ドラゴン・リッチ | 精鋭PCでも苦戦。戦術・準備が前提 |
| S | エルダードラゴン・六星クラス | 単体PCが正面から勝つことは想定外 |

> **難易度ハード原則** — 油断・準備不足・情報不足は死に直結する。  
> モンスターは「倒す対象」ではなく「状況の一要素」として扱うのが基本。
