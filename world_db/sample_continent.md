# サンプリュー大陸

> World Database。周回をまたいで変わらない正史。
> 粒度の原則：支配構造・くすぶる緊張・トーンを持つ。詳細な年表・家系図は持たない。
> 各勢力の個別NPC詳細は `world_db/notable_npcs.md` を参照。

---

## 大陸概観

サンプリュー大陸は、北の魔王国から南の海洋国まで、人類・モンスター・廃墟・海が重なり合う複合的な大陸である。大陸を南北に貫く最大の緊張軸は「人類対魔王国」だが、人類陣営は一枚岩ではなく、内部矛盾と利権争いが絶えない。

**地理（北から南へ）**

```
ゾヴェルディーナ魔王国  ← 最北端。モンスターの国家
ヴェレン               ← 主なき荒廃地帯
ヨプの大森林           ← 天然の障壁。南北を隔てる
アルカディア王国  ノルトハイム帝国  ← 東西に並立する人類国家
ポートリア海洋国        ← 最南端の半島
```

## A. アルカディア王国

**所在：** 大陸中央南部。ヨプの大森林の南側に広がる農耕・信仰文化の王国。

**支配構造：**
国王が頂点に立つが、病に伏しており統治が停滞している。宰相が実権を握りつつあり、腐敗が進行中。王族・大貴族・下級貴族による封建制。アルカディア医療教会が治癒魔法・神託の儀・孤児院・聖女候補教育を独占し、王権と協力しながらも緊張関係にある。王立魔術学院（フラニエが制度の代表格）が魔法研究・鑑定士育成を担う。冒険者ギルドが辺境実務を担当する半公的組織として機能する。

**王家の現状：**
- 国王：老病。伝統と安定の象徴。悪王ではないが停滞の源。
- 第一王子：放蕩の名が高いが剣技は優秀。自ら王に向かないと自覚し、即位を避けたがっている。
- 第二王女：聡明な改革派。政治的手腕に秀でる。現在、何者かに狙われている気配がある。
- 宰相：腐敗と汚職の中核。王の病を利用して権力を集中させつつある。

**王都アルカディア：** 「白亜の聖都」として知られる。王城・大聖堂・魔術学院・冒険者ギルド・商業区が並立する。しかし白亜の大通りを外れれば、下町・闇市・地下水道・無認可治癒師が息づく。美しいが清浄ではない。

**北方アルカディア（辺境）：** ヨプの大森林の影響を直接受ける辺境地。ゴブリン・オーク・野生モンスターの脅威が日常的。中央の目が薄く、犯罪や禁術が発覚しにくい土地。辺境領主・村自警団・巡回神官・冒険者ギルド北方支部が複合的に秩序を保っている。

**くすぶる緊張：**
- 王権 vs 教会：治癒魔法・神託の儀を誰が主導するか
- 教会 vs 学院：治癒魔法を秘匿するか理論化・公開するか
- 商会 vs 教会：医療魔道具の市場化をめぐる独占争い
- 宰相の腐敗と王家の弱体化
- 第二王女への脅威
- 魔王国への警戒（北方辺境）

**トーン：** 王道と腐敗が同居する。外見は荘厳な聖都、内側はグレーの政治。辺境は素朴で不穏。

## B. ノルトハイム帝国

**所在：** 大陸中央南部。アルカディア王国の東隣に位置する軍事帝国。

**支配構造：**
皇帝を頂点とする厳格な階層社会。軍閥貴族・官僚機構・国家教団の三本柱が国家を支える。実力と規律を価値の中心に置き、魔法文化より軍事力を優先する。

**くすぶる緊張：**
- 魔王国への敵愾心は強いが、現状は大規模開戦に至っていない
- 魔王国がなければアルカディア王国とも争っていたと言われる拡張主義的な国柄
- ヴェレンへの進出意欲を持ち、勢力圏拡大を狙っている
- アルカディア王国との表面的な協調と水面下の対立

**トーン：** 合理主義と帝国主義。規律と抑圧。戦時体制の張り詰めた空気。

## C. ゾヴェルディーナ魔王国

**所在：** 大陸最北端。ヨプの大森林とヴェレンの向こう側。魔大陸とも呼ばれる。首都ゾルヴァは魔王城を中心に広がる、大陸最大のモンスター集積地。魔王城は滅亡戦争の最大の戦場跡の上に立ち、マナ濃度が大陸で最も高い地点に位置する。

**支配構造：**
魔王の絶対王政。完全実力主義。多種族混成。魔王の意志が最終的な法であり、日常の秩序は実力者（六星・地方有力者・軍団長）が維持する。紛争は決闘で解決されることが多い。魔王への忠誠は政治的・社会的義務を兼ねる。

魔王の代替わりは純粋な実力制——最強の者が次の魔王となる。固定的な世襲制はない。

**六星（最高幹部）：**
魔王が任命する六人の最高幹部。各自が独自の拠点・軍団を持つ。詳細は `world_db/notable_npcs.md` を参照。

| 称号 | 種族 | 拠点 | 役割 |
|---|---|---|---|
| 諸刃のガエン | 人狼 | 北部・凍てつく荒野の砦 | 獣人系戦闘部隊の将 |
| 反転のギギ | エルダー・リッチ | 西部・黒い湿原の研究塔 | 魔法・死霊術研究 |
| 色慾のネモフィラ | サキュバス | 中央・魔王城下の館 | 諜報・暗殺 |
| 狡知のゲヘナ | 悪魔 | 魔王城内の塔 | 内政・参謀 |
| 堕姫のシンシア | ダークエルフ | 南部の古い砦 | 騎士団長・正面戦 |
| 第六席 | 不明 | （前任者の廃墟） | 空位（随時設定） |

**第六席について：** 物語開始時に死亡している。前任者の拠点は廃墟または別の六星が仮管理。この枠は周回ごとに自由に設定してよい。

**ウロコのないドラゴン：** 魔王と直接関係を持つとされる謎の存在。六星とは別系統。詳細不明。

**くすぶる緊張：**
- 魔王の正体は大陸最大の謎のひとつ。噂は錯綜し、誰も真実を知らない
- 六星内部の権力争い。第六席の空位をめぐる動き
- ネモフィラによる魔王への誘惑（進展していない）
- ゲヘナと魔王の「契約関係」の噂
- ポートリア海洋国との水面下の取引（ゲヘナが管轄との説あり）
- 人類陣営への本格侵攻をいつ再開するか

**トーン：** 弱肉強食。多種族が入り乱れる混沌。絶対権力の圧政と野望の渦。

## D. ヴェレン（主なき土地）

**所在：** ヨプの大森林の北側、魔王国の南側に広がる広大な荒廃地帯。地名であり国家ではない。地理は沼地・雪原・荒野・森林が主の不毛地帯。

**歴史的経緯：**
約150年前、魔王国の西方遠征によって当地の複数国家が崩壊した。10年後に魔王国は撤退したが、撤退時に略奪を尽くしたため土地は復興しなかった。以降、支配者不在の群雄割拠状態が続く。住民は苦しくも住み着いており、盗賊団・遊牧氏族・小国家・隠れ里が点在する。

**主要勢力（例）：**

**山賊王朝**
山賊集団が住民保護と引き換えに税を徴収する構造が拡大し、疑似国家的な秩序を形成した。法体系は整わないが、「呪術師に支配されるよりはまし」として住民に比較的受け入れられている。

**黄金と沼の国ウラリス**
東部湿地帯の秘境都市国家。巨大キノコと底なし沼が広がる「黄金の沼」に首都を置く。「黄金の魔術」（光・雷操作・変身魔術）と呪術（闇炎・呪い付与）の独自体系を持つ。閉鎖的で外部接触を限定するが、自国の独立と影響圏は確実に維持する。帝国の浸透工作には静観を装いながら、水面下で情報収集と独立維持の備えを続けている。

**メジプト砂炎国**
南西部砂漠のオアシスを軸とする首長国。砂魔術・炎魔術と機動的な軽騎兵で知られる。隊商路を通じて西方諸国と繋がる。帝国の進出を表向きは「共通の脅威への対抗」として歓迎しつつ、実際には帝国の混乱を好機として自国の影響圏拡大を狙っている。

**くすぶる緊張：**
- 強力なモンスターと魔王国残党の出没
- 諸勢力間の利権争いと不安定な均衡
- ノルトハイム帝国の浸透——工作員・資金・親帝国派の台頭が群雄割拠の均衡を崩しつつある
- 帝国の圧力を前に、各小勢力が「服従・抵抗・逃亡・利用」のいずれを選ぶかという岐路
- 荒廃した土地に眠る過去の遺構と資源

**トーン：** 荒廃と生存。法なき世界の活力と残酷さ。終末的だが人の営みが息づく。

## E. ポートリア海洋国

**所在：** 大陸最南端の温暖な半島。長い海岸線となだらかな山脈を持つ。

**支配構造：**
首都を持たない複数港湾都市の連合国家。商人ギルド連合が連邦的に運営する。外交・交易で生存する国であり、魔王国とも取引交渉を行う唯一の人類国家として知られる。

**くすぶる緊張：**
- 魔王国との交易：戦時景気と戦争リスクが常に背中合わせ
- 各港湾都市間の利権争い
- 人類陣営から「裏切り者」と見られるリスクへの綱渡り
- ゲヘナ（魔王国）との水面下の取引の存在

**トーン：** 猥雑で活気。金と情報が動く。道徳より損得。

## ヨプの大森林

**性格：** 大陸を南北に隔てる天然の障壁。アルカディア王国の北方辺境に接する。

- エルフが樹上で暮らす深い森。浅層は冒険者ギルドの活動域。中層から奥は未踏・危険地帯。
- ゴブリン・オーク・各種モンスターの発生源。北方アルカディアへの脅威の震源地。
- 森の変異・マナ異常が定期的に発生する。
- 薬草・木材・魔獣素材など豊富な資源を持つ。

## 固定ワールドクエスト

GMはサンプリュー大陸を舞台とする場合、以下のワールドクエストを大陸の背景として持つ。PCが関与しなければ、「不介入時の結末」へと自走する。

### WQ-01「王冠の暗夜」

**関連勢力：** アルカディア王国

**WQ起点での舞台設定（自動取得）**
```
支配者     ：腐敗した宰相の影響下にある王国（王は病床）
緊張       ：王女への脅威、権力集中、信仰と政治の癒着
トーン     ：陰謀と退廃（王都）/ 素朴で不穏（辺境）
```
プレイヤーが記入するのは「土地の詳細」（王都のどの区か、辺境の村かなど）と「PCの所在理由」のみ。PCはこのWQと無関係な人物でよい。

**GM占有：真相（世界の住民は知らない）**
王女暗殺を企てているのは宰相と教皇の結託である。宰相は王女の改革が自身の利権を破壊すると読み、教皇は改革派が台頭すれば教会の治癒魔法独占が崩れると恐れる。両者の利害が一致した。暗殺者の侵入ルートは王都地下水道を経由して王城区に抜ける経路が計画されている。

---

**タイムライン（PCが関与しなければこう進行する）**

```
【現在】
  国王が病床にある。意識はあるが政務はとれない。
  宰相が行政の実権を握り、官僚機構を通じて王女の改革案を妨害中。
  教皇が王子への接近を続けている。
  暗殺計画は実行者の選定が完了し、日程を調整中。
  侵入ルート（地下水道）の下見が行われている。

【フェーズ1 ── 数週間後】
  王女の側近が買収または脅迫され、孤立化が完成する。
  宰相が医療教会への物資供給を人質に、教皇の全面協力を取り付ける。
  暗殺者が地下水道の下見を終え、実行準備が整う。

【フェーズ2 ── 暗殺実行】
  暗殺者が地下水道から侵入し、王女を殺害する。
  死因は「急病」として処理される。教会が「手当てが間に合わなかった」と証言し、
  宰相が即座に情報統制を敷く。

【フェーズ3 ── 余波】
  王女の死を知った国王が急速に衰弱する。
  教皇が王子への働きかけを強め、王子は追い詰められて即位を渋々受け入れる。

【フェーズ4 ── 数ヶ月後・新王即位】
  国王崩御。王子が即位するが、宰相と教皇が連立して実権を握る。
  改革派の貴族・商会は弾圧または懐柔される。
  北方辺境の対応が後回しになり、モンスター被害が拡大し始める。

【フェーズ5 ── 長期】
  腐敗と停滞が深化する。
  ノルトハイム帝国が「支援」を名目にアルカディアへの影響力を伸ばす（WQ-04に連動）。
  王国は表向き安定しながら実質的に形骸化していく。
```

---

**PCが関与しうるポイント（確定ではなく可能性の羅列）**

PCがどこで何をするかによって、タイムラインは分岐・停止・遅延・変質する。
以下はその一部であり、GMはPCの行動から自然に発生する介入を認める。

- **直接阻止** — 暗殺計画を察知し、王女を守る。計画を暴露する。
- **犯人の排除** — 宰相または教皇を先に始末する。計画は頓挫するが、別の権力者が台頭する可能性がある。
- **逆に利用する** — 宰相・教皇を保護し、暗殺を成功させて腐敗を加速させることで、別の目的（混乱の利用、王国の弱体化など）を達成する。
- **第三勢力として動く** — 王国の混乱を好機と見て、征服・支配・傀儡化を企てる側として関与する。この場合も、WQ-01はある意味「解決」しうる。
- **不介入** — 原則。タイムラインは自走する。PCは結果の観測者になる。
- **バタフライ効果** — PCが王女暗殺と無関係に行動した結果として、タイムラインが変質することがある。

> バタフライ効果の例：
> ネズミのPCが王都の地下水道を縄張りとして支配した結果、暗殺者の侵入ルートが封鎖される。
> PCは王女暗殺を知らない。しかし暗殺は失敗し、計画は再調整を余儀なくされ、フェーズ2が遅延または変質する。
> このような、PCの意図しない波及は正規の介入として扱う。

### WQ-02「六星の影が南へ」

**関連勢力：** ゾヴェルディーナ魔王国、ヴェレン、アルカディア王国（辺境）

**WQ起点での舞台設定（自動取得）**
```
支配者     ：辺境領主または無政府（ヴェレン）
緊張       ：森から来る何か・魔物増加・見知らぬ工作員の出没
トーン     ：素朴で不穏（辺境）/ 荒廃と生存（ヴェレン）
```
プレイヤーが記入するのは「アルカディア辺境かヴェレンか」の選択と「PCの所在理由」のみ。

**現在地点：**
魔王国の六星のひとり（ゲヘナ、あるいは第六席の後任）がヴェレンに足がかりを作ろうと動いている。前哨作戦として、ヴェレン内の勢力への工作と、アルカディア辺境への陽動が進みつつある。

**不介入時の結末：**
ヴェレンに魔王国の前線拠点が確立される。アルカディア辺境のモンスター活動が増加し、住民・冒険者の被害が拡大する。ノルトハイム帝国が事態を利用してヴェレンへの介入を正当化する。

**PCが関与できるポイント：**
辺境の異変を最初に観測する、工作員の動きを追う、ヴェレンの小勢力と接触して情報を得る、前線拠点の建設を妨害または見過ごす。

### WQ-03「帝国の楔」

**関連勢力：** ノルトハイム帝国、ヴェレン（山賊王朝・ウラリス・メジプト）

**WQ起点での舞台設定（自動取得）**
```
支配者     ：山賊王朝・ウラリス・メジプトのいずれか、または無政府地帯
緊張       ：外来勢力の浸透・旧秩序の崩壊・住民の選択
トーン     ：荒廃と生存・終末的
```
プレイヤーが記入するのは「ヴェレンのどの地域か」と「PCの所在理由」のみ。

**現在地点：**
ノルトハイム帝国の工作員・資金・代理人がヴェレン各地に浸透しつつある。親帝国派の小勢力や商人が台頭し、150年続いた群雄割拠の均衡が静かに崩れ始めている。山賊王朝は帝国への服従を拒んで武力衝突の瀬戸際にあり、ウラリスは沈黙を守り、メジプトは帝国を利用しようとして逆に取り込まれかけている。

**不介入時の結末：**
帝国がヴェレン西部から北部にかけての支配を確立する。山賊王朝は解体されるか従属化し、住民は帝国の秩序——搾取ではあるが混乱よりはましな秩序——に組み込まれる。ウラリスのみが湿地の奥に引きこもり独立を保つ。ヴェレンは「主なき土地」から「帝国の辺境」へと変わる。

**PCが関与できるポイント：**
帝国の工作員を発見する、山賊王朝の抵抗を助けるか妨げる、ウラリスやメジプトの選択に関与する、帝国支配の恩恵を受ける側に立つ、住民の変化を観測する。

### WQ-04「二つの王冠の間で」

**関連勢力：** ノルトハイム帝国、アルカディア王国

**WQ起点での舞台設定（自動取得）**
```
支配者     ：名ばかりの領主（帝国の影響下に入りつつある辺境貴族）
```
