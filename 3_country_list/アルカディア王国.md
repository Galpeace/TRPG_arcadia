# 第1部　国家概観

## 1.1 基本情報

- 国名：アルカディア王国
- 位置：アルカディア大陸北東部、ヨプの大森林に環状に囲まれた盆地
- 地形：肥沃な平野と盆地
- 気候：温帯
- 統治形態：王制
- 主要拠点：王都カディア
- 国柄：魔法文化が盛んな大国。冒険者ギルドの本部と魔法学院を擁する

## 1.2 国の性格

ヨプの大森林という天然の防壁に守られた内陸国家です。外敵の侵入は森林越えを要するため、王国は長い歴史を通じて比較的安定した秩序を維持してきました。その安定が魔法研究と冒険者文化の発展を支えた一方で、外の世界への危機感の薄さも生んでいます。魔王国を「対岸の火事」として認識しているのは、その地理的条件が大きく影響しています。

## 1.3 現在の政治状況

国王ハインリヒ（54歳）が即位22年。安定と伝統を重んじる現状維持型の統治です。宰相アルブレヒトの汚職を黙認し、改革派王女リリアとは軋轢があります。東海岸確保のための森林伐採事業が現在の最大の政治的課題です。

# 第2部　王都カディアの地理と構造

## 2.1 概要

王都カディアは王国の中心都市であり、政治・宗教・学術・商業のすべてが集積しています。各機能が地区として分離しており、地区ごとに雰囲気と住人の層が大きく異なります。

## 2.2 王城周辺・貴族街

王城を中心とした地区で、王都の中で最も整備された区画です。

- 王城および王宮施設
- アルカディア医療教会の大聖堂
- フラニエと並ぶ魔法学院
- 貴族の邸宅が立ち並ぶ
- 清潔で秩序があり、庶民が立ち入る機会は限られる
- 衛兵の巡回が多く、治安は高い

## 2.3 庶民街

王都の住人の大多数が暮らす地区です。

- 大規模な市場が複数あり、交易の中心
- 冒険者ギルドの本部がここに置かれている
- 宿、酒場、職人工房、商家が密集する
- 活気があり騒々しい
- 多様な人々が行き交い、情報が集まる

## 2.4 貧民街

庶民街のはずれに広がる地区です。

- 経済的に困窮した住人が多い
- カディアで居場所をなくした獣人が流れ着くことが多い
- 他国からの流れ者、元罪人、異形の存在が紛れ込む
- 衛兵の巡回は少なく、自警的な秩序が機能している部分もある
- 王国の社会問題（移民排斥・経済格差）が最も可視化される場所

# 第3部　政治と社会

## 3.1 支配構造

王制のもと、宰相を頂点とする貴族・官僚機構が実務を担います。アルカディア医療教会と魔法学院が教育・医療を通じて社会に深く根ざしており、宗教・学術の両面からも影響力を持ちます。

## 3.2 社会的摩擦

王国内には解決されていない社会問題が存在します。

**移民排斥**

獣人・異形・他国出身者への差別的な扱いが慣行化しています。法的な保護は限定的で、貧民街がその受け皿となっています。改革派のリリア王女はこれを問題視していますが、現状は改善されていません。

**経済格差**

貴族・商人層と庶民・貧民層の格差は大きく、宰相アルブレヒトの汚職がこれをさらに悪化させています。

## 3.3 対外関係の認識

ポートリア海洋国・カタリナ王国と同盟。エマーシア王国とは魔法文化をめぐる対立。魔王国は地理的に遠く、脅威として現実的に認識されていない。これがノルトハイム帝国との世界認識の最大の差です。

# 第4部　冒険者ギルドと王国軍

## 4.1 王国軍

王国の正規軍事力です。国境警備・街道の安全確保・国王の命による遠征などを担います。貴族出身の将官が上位を占め、軍閥的な性格も持ちます。

## 4.2 冒険者ギルド

庶民街に本部を持つ半傭兵的な組織です。

- 王国軍の補完的存在として機能する
- 王国が直接対処しにくい案件（森林内の魔物討伐、民間依頼の達成など）を担う
- 独立した組織であり、王国軍とは指揮系統が別
- 依頼主は王国・貴族・商人・一般市民と幅広い
- 冒険者の質はピンキリで、Sランクから駆け出しまでが混在する
- ギルド長ガルドは「竜狩り」の異名を持つ実力者

## 4.3 両者の関係

協力関係にありますが、利害が一致しない場面もあります。王国軍は冒険者を「御しにくい連中」と見なしており、冒険者側も王国軍の官僚的な体質を嫌う気風があります。ただし実質的な補完関係は成立しており、大きな衝突には至っていません。

# 第5部　物語類型と運用上の指針

## 5.1 アルカディア王国を舞台にした物語類型

**1. 王都政治の物語**

- 宰相汚職の暴露
- リリア王女の改革派と保守派の対立
- 後継者問題への関与
- 貴族間の権力争い

**2. 社会問題の物語**

- 貧民街での出来事
- 獣人・異形への差別と向き合う
- 経済格差の中での生存

**3. 冒険者の物語**

- ギルドの依頼をこなす日常
- 王国軍との協力・軋轢
- 大型魔物討伐

**4. 宗教・学術の物語**

- 医療教会の内部事情
- 魔法学院での研究と派閥
- 邪神崇拝との対峙

## 5.2 PCの想定類型

- 冒険者
- 王国軍の兵士・将校
- 貴族・貴族の使用人
- 教会関係者・魔法使い
- 商人・職人
- 貧民街の住人・流れ者
