# 北方アルカディア設定書 このファイルは北方アルカディアの設定をまとめたものです。世界設定書（世界設定書.md）の第5部「アルカディア王国」のうち、北方アルカディアの部分を補強します。 ## 重要な注意 **本設定書はあくまで参考要素です。** - 設定にない場所・勢力・人物をGMが自由に導入してよい - 設定にある要素を必ず登場させる必要はない - 各要素の細部は固定的ではなく、シナリオに応じて調整可能 - プレイ内容に応じて本設定書は事後的に拡張・改変される ---- ## 目次 1. 地域概観 2. 主要集落 3. 暮らしと産業 4. 主要勢力（緩やかな) 5. 物語類型と運用上の指針 6. 関連ファイル ---- # 第1部　地域概観 ## 1.1 基本情報 - 名称：北方アルカディア - 位置：アルカディア王国の北方辺境地帯 - 地形：複合地形（低山地、森林の縁、河川、丘陵) - 辺境の先：未踏地と大山脈などの自然の障壁 ## 1.2 気候 - 寒冷 - 冬は厳しい雪に閉ざされる - 春から秋にかけての労働の集中 - 夏は穏やかな避暑地 - 季節の循環が明確で、暮らしのリズムを作る ## 1.3 雰囲気 - 牧歌的な田舎の風景 - 「象徴的要素のなさ」が象徴的要素 - 派手な事件ではなく、日々の労働と暮らしが中心 - ただし表面の下に、異形が潜伏する余地もある - 圧力の少ない、自由な空気 ## 1.4 王都との繋がり - 街道は「ヨプの大森林」を通って王都カディアに一本で繋がる - やや危険な旅路と見なされている - 馬車での移動は数日〜一週間程度 - 経済的にはカディアに依存しつつも、文化的には独立気風 ## 1.5 社会の性格 - 貴族不在（こんな辺境に領地を持つ意味はないとされる) - 集落の自治 - 多様な層の混合（先住民、開拓民、元罪人、自由人、異形、他種族) - カディアの干渉を嫌う独立気風 - 牧歌的で素朴 ---- # 第2部　主要集落 ## 2.1 セントラリア **性格** - 街道沿いの中心集落 - 三集落の中で最も発展している - 王都からの旅人が最初に到着する場所 **機能** - 宿、交通の要衝 - 冒険者ギルド支部（小規模) - 市場 - 三集落間の物流の中継地 **規模** - 北方アルカディアでは最大、人口数千程度 - それでもカディアに比べれば小さな町 **雰囲気** - 旅人と地元民が交錯する - 街道沿いの活気と、辺境の素朴さが共存 - 様々な噂が集まる場所 ## 2.2 ヨウィン **性格** - 畜産業・農業の中心 - 米の名産地 - 伝統的な農村集落 **機能** - 食料生産の中心 - 季節の労働と暮らしが営まれる場所 - 北方アルカディア全体の食料供給を支える **規模** - 人口数百〜千程度 - 周辺に小さな農村が点在 **雰囲気** - 牧歌的、田園的 - 季節の循環に従った生活 - 伝統的な農村文化 - 米作りの祭、収穫祭などの行事 ## 2.3 ヴェルニース **性格** - 鉱業・林業の中心 - 山岳寄りに位置 - やや粗野な雰囲気 **機能** - 鉱物（鉄、銀、稀少な鉱石など)の採掘 - 林業（木材、薪、樹液など) - 狩猟の拠点 **住人** - 鉱夫、樵、狩人 - 厳しい労働で鍛えられた人々 - 比較的単身者が多い、流れ者も含む **雰囲気** - 粗野、たくましい - 酒場が中心の社交場 - セントラリアやヨウィンとは異なる気風 ---- # 第3部　暮らしと産業 ## 3.1 主要産業 - 農業（米が美味で有名) - 畜産業（牛、羊、鶏など) - 鉱業（ヴェルニースが中心) - 狩猟 - 林業 - 冒険者業（小規模) ## 3.2 経済 - 物々交換ベース - 貨幣経済も成立しているが、日常的には現物のやり取りが多い - 王都との交易は限定的（街道の危険、輸送コスト) - 地域内での自給自足的な経済 ## 3.3 季節の循環 **春** - 雪解けと共に農作業が始まる - 種まき、家畜の出産 - 街道の通行が再開される **夏** - 農作業の繁忙期 - カディアからの避暑客が訪れる（一部の有力者) - 狩猟、林業も活発 **秋** - 収穫期 - 米、野菜、家畜の屠殺 - 冬への蓄え - 収穫祭 **冬** - 雪に閉ざされる - 室内での手仕事、家畜の世話 - 街道の通行は困難 - 集落間の交流が減る ## 3.4 文化 - 素朴な民俗文化 - 古い民謡、踊り - 季節の祭り - 物語の口承 - 信仰は穏やか（土着の神々、聖女セシリアへの遠隔的な信仰など) ## 3.5 食文化 - 米が中心 - 家畜の肉、乳製品 - 狩猟で得た野生肉 - 山菜、キノコ - 寒冷地特有の保存食（漬物、燻製、発酵食品) ---- # 第4部　主要勢力（緩やかな) 北方アルカディアには明確な「勢力」は存在しません。集落の自治と、緩やかな共同体が中心です。ただし、以下のような緩やかなグループ分けは存在します。 ## 4.1 集落の長老・有力者 各集落には自然と長老や有力者が存在します。具体的な人物は 北方アルカディアキャラクター.md を参照。 ## 4.2 地域全体の評議会（緩やか） 集落間で問題が起きた時、各集落の代表が集まる緩やかな評議会が形成されます。常設ではなく、必要に応じて開かれる。 ## 4.3 流入者・流れ者 北方アルカディアには様々な流入者がいます。 - カディアでは生きにくい者 - 元罪人、追放者 - 異形・人外の存在 - 他種族（エルフ、ドワーフ、ハーフリング) - 旅人、冒険者 これらは集落に溶け込むこともあれば、外縁部で独自の生活を送ることもある。 ## 4.4 隠れた存在 牧歌的な表面の下に、異形が潜伏する余地があります。 - 山中で密かに暮らす存在 - 集落内で「普通の住人」を装う異形 - 過去のプレイ傾向（変質する豚、邪悪な術者など)が活躍する余地 これらは設定として明文化されない潜在的要素であり、シナリオに応じて運用されます。 ## 4.5 王国軍・冒険者ギルドとの関係 - 王国軍の常駐はない - 冒険者ギルドのセントラリア支部のみが存在 - カディアからの干渉は最低限 ---- # 第5部　物語類型と運用上の指針 ## 5.1 北方アルカディアを舞台にした物語類型 **1. 牧歌的な日常の物語** - 季節の循環の中での暮らし - 集落内の人間関係 - 労働と祭、収穫と冬越し - 派手な事件はないが、深い人間ドラマ **2. 地域社会の問題解決の物語** - 集落間の小さな揉め事 - 鉱山事故、天候不順、家畜の病気 - 行方不明者の捜索 - 共同体の絆 **3. 異形の居場所探しの物語** - 流入してきた異形の存在 - 集落への適応・対立・受容 - 静かな受け入れと、突発的な拒絶 - 過去のプレイ傾向と強く整合 **4. 表と裏の二重性の物語** - 牧歌的な表面の下の異変 - 集落の隠された秘密 - 山中に潜む存在 - 集落の中に紛れ込んだ異形 **5. 街道の物語** - 街道沿いでの事件 - 王都からの旅人との出会い - 街道の魔物・盗賊との対峙 - 行方不明の隊商 **6. 季節と労働の物語** - 米の収穫 - 鉱山での労働 - 狩猟の季節 - 冬越しの準備 ## 5.2 PCの想定類型 北方アルカディアを活動拠点とするPCは、以下のような立場が想定されます。 - 地元の農夫、職人、狩人、鉱夫 - 集落の若者 - カディアから流れ着いた者 - 異形の存在（変質する豚、知性ある猫、邪悪な術者など) - 元冒険者、元軍人、元罪人 - 流れの旅人、隠者 - 他種族（エルフ、ドワーフ、ハーフリング) 北方アルカディアは圧力の少ない自由な土地として、過去のプレイ傾向（異形・周縁的キャラクター）に対して特に居場所が多い場所です。 ## 5.3 物語の重要な軸 **労働と季節の循環** - 季節の変化が物語のリズムを作る - 労働は背景ではなく、物語の中心になりうる **素朴な人間ドラマ** - 派手な事件ではなく、人間関係の機微 - 共同体の温かさと閉鎖性 - 受け入れと排除 **表と裏の二重性** - 牧歌的な表面と、潜伏する異形 - 平和な日常と、突発的な異変 **カディアとの距離** - 王都とは別の論理で動く土地 - 中央の腐敗から相対的に独立 - 独立気風と素朴さ ---- # 第6部　関連ファイル 北方アルカディアの設定は以下のファイルに分散して保管されます。 - 北方アルカディア設定書.md：本ファイル、北方アルカディアの地理・勢力・運用指針 - 北方アルカディアキャラクター.md：北方アルカディアの重要人物 - 魔物魔族リスト.md：魔物・魔族リスト すべてのファイルは参考要素として機能し、シナリオ運用時の創造的自由を制限しません。 ---- # 補遺：北方アルカディアの独自性について 北方アルカディアの最大の独自性は、「象徴的要素のなさ」です。ダンジョンもなく、強大な独立勢力もなく、特殊な生態系もなく、神秘もない。ただ広がる農地と牧場、点在する集落、低山地と森林の縁。 これは多くのファンタジー作品の辺境と一線を画します。「冒険の地」ではなく「生活の地」としての辺境。 しかし圧力の少ない場所であるがゆえに、異形の存在が密かに勢力を蓄える舞台にもなりえます。表面の牧歌性が強いほど、裏の異変の不気味さが際立ちます。 この二重性を活かした物語が、北方アルカディアでは可能になります。 - 表の物語：牧歌的な地域社会の暮らし - 裏の物語：表面の下で進行する異変、潜伏する異形 両者は対立せず、共存します。シリーズによって、表面のみを描く穏やかな物語、裏面に踏み込む不穏な物語、両方を交錯させる重層的な物語、いずれも展開できます。
