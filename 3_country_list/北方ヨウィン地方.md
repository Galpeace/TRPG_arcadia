# 第1部　地域概観

## 1.1 基本情報

- 名称：北方ヨウィン地方
- 性格：アルカディア王国にもフェルガード連邦にも属さない、辺境の地理的・文化的総称
- 位置：大陸最北東、ヨプの大森林の北側
- 隣接：南にヨプの大森林、西にフェルガード連邦、北は海および未踏の開拓地
- 地形：なだらかな平原と山地の混在
- 気候：寒冷
- アクセス：ヨプの大森林を越える街道（困難）、または北方海岸からの船便

## 1.2 独立性の根拠

北方ヨウィンがアルカディア王国から独立した地域として存在できているのは、ヨプの大森林がその間に横たわっているためです。森林は外敵からアルカディアを守る防壁であると同時に、アルカディアの支配がヨウィンへ及ぶことを妨げる障壁でもあります。交易コストも高く、アルカディア側にヨウィンを統治するメリットがありません。

この結果、両者は互いにほとんど干渉しない関係にあります。アルカディアはヨウィンを版図と見なしておらず、ヨウィン側もアルカディアへの帰属意識を持っていません。

## 1.3 フェルガード連邦との関係

西隣のフェルガード連邦（獣人族の国）とは、長年の慣習的友好関係にあります。条約や同盟といった文書的な裏付けはなく、世代を超えた付き合いの蓄積がその基盤です。ヨウィンは獣人を排除せず、カディアで居場所をなくした獣人が流れ着くこともあります。フェルガード側もヨウィンを数少ない信頼できる隣人と見なしており、ヨウィンが脅かされれば動くでしょう。ただしそれは義務ではなく、感情に基づくものです。

## 1.4 気候と季節

- 寒冷
- 冬は厳しい雪に閉ざされる
- 春から秋にかけての労働が集中する
- 夏は穏やかな短い季節
- 季節の循環が暮らしのリズムを作る

## 1.5 社会の性格

- 統治機構を持たない。集落の自治が基本
- 集落間の問題が起きた時のみ、各集落代表が集まる緩やかな評議が行われる（常設ではない）
- 先住民・開拓民・流れ者・獣人・異形など多様な層が混在する
- 貴族が存在しない（辺境の地に領地を持つ意味がないとされる）
- 北方の未踏地への開拓が緩やかに進んでいる

# 第2部　主要集落

## 2.1 セントラリア

**性格**

- 街道沿いの中心集落
- 三集落の中で最も発展している
- 森林を越えてくる旅人・商人が最初に到着する場所

**機能**

- 宿、交通の要衝
- 冒険者ギルド支部（小規模）
- 市場
- 三集落間の物流の中継地

**規模**

- 北方ヨウィン地方では最大、人口数千程度

**雰囲気**

- 旅人と地元民が交錯する
- 街道沿いの活気と、辺境の素朴さが共存
- 様々な噂が集まる場所

## 2.2 ヨウィン

**性格**

- 畜産業・農業の中心
- 米の名産地
- 伝統的な農村集落

**機能**

- 食料生産の中心
- 地方全体の食料供給を支える

**規模**

- 人口数百〜千程度
- 周辺に小さな農村が点在

**雰囲気**

- 牧歌的、田園的
- 季節の循環に従った生活
- 米作りの祭、収穫祭などの行事

## 2.3 ヴェルニース

**性格**

- 鉱業・林業の中心
- 山岳寄りに位置

**機能**

- 鉱物（鉄、銀、稀少な鉱石など）の採掘
- 林業（木材、薪、樹液など）
- 狩猟の拠点

**住人**

- 鉱夫、樵、狩人
- 比較的単身者が多い、流れ者も含む

**雰囲気**

- 粗野、たくましい
- 酒場が中心の社交場

# 第3部　暮らしと産業

## 3.1 主要産業

- 農業（米が美味で有名）
- 畜産業（牛、羊、鶏など）
- 鉱業（ヴェルニースが中心）
- 狩猟
- 林業
- 冒険者業（小規模）

## 3.2 経済

- 物々交換ベース
- 貨幣経済も成立しているが、日常的には現物のやり取りが多い
- アルカディアとの交易は限定的（森林越えの困難さとコストによる）
- 地域内での自給自足的な経済

## 3.3 季節の循環

**春**

- 雪解けと共に農作業が始まる
- 種まき、家畜の出産
- 街道の通行が再開される

**夏**

- 農作業の繁忙期
- 狩猟、林業も活発

**秋**

- 収穫期
- 米、野菜、家畜の屠殺
- 冬への蓄え
- 収穫祭

**冬**

- 雪に閉ざされる
- 室内での手仕事、家畜の世話
- 街道の通行は困難
- 集落間の交流が減る

## 3.4 文化

- 素朴な民俗文化
- 古い民謡、踊り
- 季節の祭り
- 物語の口承
- 信仰は穏やか（土着の神々への信仰が中心）

## 3.5 食文化

- 米が中心
- 家畜の肉、乳製品
- 狩猟で得た野生肉
- 山菜、キノコ
- 寒冷地特有の保存食（漬物、燻製、発酵食品）

# 第4部　主要勢力（緩やかな）

## 4.1 集落の長老・有力者

各集落には自然と長老や有力者が存在します。集落の日常的な判断はここで行われます。

## 4.2 地域全体の評議（緩やか）

常設の評議会は存在しません。集落間で問題が起きた時のみ、各集落の代表が集まって話し合います。

## 4.3 流入者

北方ヨウィン地方には様々な流入者がいます。

- カディアで居場所をなくした者
- 元罪人、追放者
- 獣人（フェルガード連邦からの移住者、カディアからの流入者）
- 異形・人外の存在
- 旅人、冒険者

これらは集落に溶け込むこともあれば、外縁部で独自の生活を送ることもあります。

## 4.4 冒険者ギルドとの関係

- セントラリアに小規模な支部がある
- カディアのギルドとの接続は維持されているが、独立性が高い
- 地域内の依頼（魔物対処、捜索など）を主に扱う

# 第5部　物語類型と運用上の指針

## 5.1 北方ヨウィン地方を舞台にした物語類型

**1. 牧歌的な日常の物語**

- 季節の循環の中での暮らし
- 集落内の人間関係
- 労働と祭、収穫と冬越し

**2. 地域社会の問題解決の物語**

- 集落間の小さな揉め事
- 鉱山事故、天候不順、家畜の病気
- 行方不明者の捜索

**3. 異形・流れ者の居場所探しの物語**

- 流入してきた者が集落に溶け込む過程
- 受け入れと摩擦
- 独自の生き方を模索する

**4. 街道と森林の物語**

- 森林越えの護衛
- 街道上の魔物・盗賊との対峙
- カディアとヨウィンをつなぐ旅

**5. フェルガードとの関係の物語**

- 獣人住民との共同生活
- フェルガード連邦との国境付近での出来事

## 5.2 物語の重要な軸

**労働と季節の循環**

季節の変化が物語のリズムを作ります。労働は背景ではなく、物語の中心になりえます。

**素朴な人間ドラマ**

派手な事件ではなく、人間関係の機微から物語が生まれます。共同体の温かさと閉鎖性が共存しています。

**カディアとの距離**

森林によって実質的に断絶しているため、王都の論理が直接及ばない土地です。中央の政治とは別の原理で動いています。

**獣人との共生**

フェルガードとの慣習的友好を背景に、獣人が普通に暮らしている地域です。これはカディアの貧民街とは異なる、差別を前提としない関係です。

## 5.3 PCの想定類型

- 地元の農夫、職人、狩人、鉱夫
- カディアから流れ着いた者
- 獣人（フェルガードからの移住者含む）
- 元冒険者、元軍人
- 流れの旅人
