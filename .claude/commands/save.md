description: セッションをセーブする（ログ書き出し + コミット）
---
現在のセッションをセーブする。

1. 今回のプレイ内容を sessions/YYYY-MM-DD.md に書き出す（同日2回目以降は追記）。
   内容: 日付、参加キャラ、出来事の要約、重要な判定とその結果、
   獲得/喪失したもの。
2. ログ末尾に「## 次回への引継ぎ」を3〜5行で書く
   （現在地・直近の目的・未解決の出来事・次の場面の入り）。
3. state/ の内容がプレイ内容と一致しているか `python tools/state.py show` で確認する。
4. 変更を git add し、「セーブ: <一言要約>」の形式でコミットする。
