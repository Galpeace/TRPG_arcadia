# TRPG Arcadia — Witnessed 改訂第二版

Claude Code の中で遊ぶソロプレイ・ハイファンタジーTRPG。
**PCは世界に目撃される存在である。** 難易度はハード。死は正規の結末。
語りはClaudeが、帳簿と乱数はコードが担当する。

## 構成

- `rules/rulebook.md`（基本ルール・Witnessed v1）＋ `rules/revision.md`（**改訂第二版・優先**）
- 改訂の柱: ダイス裁定（2d6＋事情）／自走する世界／噂の伝播と歪み／
  封緘された真実・大エンドロール・死後伝説フェーズ・年代記／モンスタープレイ
- 周回は完全リセット制。UDは破棄され、伝説はその周回の中で語られる。
  `sessions/` の年代記だけがプレイヤーの書架に残る。

## 遊び方

| コマンド | 内容 |
|---|---|
| `/start` | 周回開始（方向性決定・キャラ作成・致死性条項固定・封緘） |
| `/endchapter` | 章末処理（8手順＋タイムライン進行＋噂の伝播） |
| `/recap` | あらすじ（PCが観測できている範囲のみ） |
| `/save` | プレイログ書き出し＋コミット |
| `/handoff` | 新チャットへの引継ぎ書生成 |
