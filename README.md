# TRPG Arcadia

Claude Code の中で遊ぶソロプレイTRPG。**Claudeが物語を、コードが数字を担当する。**

- ダイスは `tools/roll.py` が OS の乱数源で振る（GMの脳内ロール禁止）
- HP・所持金・アイテムは `state/*.json` が正で、`tools/state.py` 経由でのみ更新
- セーブは git コミット。`git log` がそのままプレイ履歴になる

## 遊び方

Claude Code をこのリポジトリで開いて:

| コマンド | 内容 |
|---|---|
| `/start` | セッション開始（状態読み込み or キャラメイク） |
| `/recap` | これまでのあらすじ |
| `/save` | プレイログ書き出し + コミット |
| `/handoff` | 次のチャットへの引継ぎ書を生成 |

コマンドを使わず普通に話しかけても、GM（CLAUDE.md）が同じ手順で動く。

## ゲーム概要（仮称『編纂者の眼』）

舞台はハイファンタジー世界アルカディア。世界は〈編纂者〉に観測されることで
存在を確定させており、観測の薄い辺境〈白紙〉は文字通りまだ書かれていない。

- **観測**: 未確定の事実はPCが観測した瞬間にダイスで確定する。
  `roll.py` の実ロールがゲーム内の「観測」そのもの。後出し変更は公理違反。
- **注視点**: 物語的な行いが〈編纂者〉の関心を集め、振り直し・事実宣言などに使える。
- **称号**: 目撃された偉業は称号として実体化し、力になる。
- **エンドロール**: セッションの締めに〈編纂者〉が物語を叙述し、評価を与える。
