# TRPG Arcadia

Claude Code の中で遊ぶソロプレイTRPG。**Claudeが物語を、コードが数字を担当する。**

- ダイスは `tools/roll.py` が OS の乱数源で振る（GMの脳内ロール禁止）
- HP・所持金・アイテムは `state/*.json` が正で、`tools/state.py` 経由でのみ更新
- セーブは git コミット。`git log` がそのままプレイ履歴になる

## 遊び方

Claude Code をこのリポジトリで開いて:

| コマンド | 内容 |
|---|---|
| `/start` | セッション開始（状態読み込み or キャラメイク） |
| `/recap` | これまでのあらすじ |
| `/save` | プレイログ書き出し + コミット |
| `/handoff` | 次のチャットへの引継ぎ書を生成 |

コマンドを使わず普通に話しかけても、GM（CLAUDE.md）が同じ手順で動く。

## ゲーム概要（仮称『万神の桟敷』・ドラフト案B）

舞台はハイファンタジー世界テアトラ。夜空の星々は神々の観覧席の灯であり、
世界は神々が観劇する舞台である。神々の脚光は強さも徳も選ばない——
**主役はネズミでも、伝説の剣豪でも、悪の死霊術師でもありうる。**

- **存在・真髄・格**: 能力値はない。「何者か」の宣言がキャラのすべて。
- **相対難易度**: 困難さは「その存在にとって」で測る。鼠と伝説に同じ物差しを当てない。
- **格差と見せ場**: 力比べでは格上に勝てない。だが搦め手と機転に格は無い。
- **喝采**: 神々が沸けば運命が撓む。善悪は問われない、退屈だけが罰される。
- **星座**: 死んだ主役は夜空に編まれる。ロストだが、その演目は永遠に残る。
- **カーテンコール**: セッションの締めに神々が批評し、二つ名と成長を与える。
